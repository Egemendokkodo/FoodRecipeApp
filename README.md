# Recipe Book App

This is a Recipe Book mobile application built using Java and Android Studio. Firebase and Retrofit have been used for data storage and retrieval.

## Features

- Users can browse through various categories of recipes
- Users can view the details of a particular recipe
- Users can add new recipes
- Users can edit existing recipes
- Users can delete recipes
- Users can search for recipes

## Installation

- Clone the repository to your local machine
- Open the project in Android Studio
- Sync the project with Gradle files
- Run the app on an emulator or a physical device

## Usage

- On launching the app, the user will be taken to the home screen
- The user can browse through various categories of recipes by clicking on the respective category icons
- The user can view the details of a particular recipe by clicking on the recipe card
- The user can add a new recipe by clicking on the 'Add Recipe' button and filling out the necessary details
- The user can edit an existing recipe by clicking on the recipe card and then clicking on the 'Edit Recipe' button
- The user can delete a recipe by clicking on the recipe card and then clicking on the 'Delete Recipe' button
- The user can search for a recipe by entering the recipe name in the search bar

## Technologies Used

- Java
- Android Studio
- Firebase
- Retrofit

## Credits

- The recipe data has been sourced from Spoonacular https://spoonacular.com/food-api/
